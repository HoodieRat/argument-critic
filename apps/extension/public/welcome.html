Argument Critic

Open the app with one click.

You do not need to find the extension icon first. Use the button below and Argument Critic will open in Chrome’s side panel for this window.

Use full-page mode instead

The side panel opens on the right edge of the browser.

If you choose full-page mode, chat still works there. Screen capture works best from the side panel.

  1. Click Open Argument Critic.
  2. If this is your first run, open Settings and choose Sign in with GitHub.
  3. Pin the extension later only if you want quicker access next time.
